# web-design-guidelines

## TL;DR

Orryx Group brand baseline: typography, colour palette, spacing scale, component patterns. Each entity (Triora, Pillarworks, Brain, Flow, repair-template) has overrides documented in their brand pack.

## Workflow

1. Confirm which entity brand applies (Triora / Pillarworks / Brain / Flow / repair / Orryx parent)
2. Read the entity brand pack (Triora-Brand-Identity-Pack-* / Pillarworks_Brand_Identity / Orryx-Visual-Identity-Concepts / etc.)
3. Use the canonical colour palette + typography for the entity
4. Spacing: 4px base; common scales 4/8/16/24/32/48/64
5. Components: prefer Orryx component library if it exists for the stack; else use entity-specific UI kit
6. Accessibility: WCAG 2.1 AA minimum; healthcare entities (Triora, Flow) push for AAA on critical flows
